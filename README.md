# 組織的基本概念 （==**選**==）
- 意義:人、結構、組織目標、流程
- 組織: 
    1. 兩個人或兩個人以上組合的群體
    2. 共同的目標
    3. 固定的結構
    4. 一定的運作流程

## 靜態面 => 組織結構
- 組織結構來研究組織的意義
- 機關權責分配
- 層級節制
- ex : 官僚體制
- 四大要素
    1. 目的
    2. 目標認同
    3. 適當的條配(專業分工)
    4. 權責合理化
## 動態面 => 人與人之互動行為
- 組織 交互行為
## 心理面 => 成員精神層面
- 制度是成長起來的，不是製造出來的 =>　行政生態學
## 生態面 => 有機的成長體
- 心理、精神
